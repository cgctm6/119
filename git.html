git init 初始化git git config --global user.name "XXX" 设置用户名 git config --global user.email "XXX@qq.com" 设置用户邮箱 git add ./XXX 添加某文件到仓库大门口(缓存区) git commit -m "XXX" 提交到git的仓库 git status 查看当前状态 git add ./ 修改过的文件全部添加到仓库大门口 git commit --all -m "XXX" 将所有的文件提交到git的仓库了 git log 查看历史提交的日志 (提交的时间作者，文件个数) git log --oneline 简介版本的查看日志了 git reset --hard head~0 表示往回走一个日志
